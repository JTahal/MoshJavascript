Dealing with a list of object
[ ] <---- empty array or array Literal
let selectedColor = [];

e.g.:

let selectedColor = ['red', 'blue'];
console.log(selectedColor);

Display first element in the array
console.log(selectedColor[0]);

Add a item to list
let selectedColor = ['red', 'blue'];
selectedColor[2] = 'green';
console.log(selectedColor);

Return the amount of items in a list
console.log(selectedColor.length);
